# 🧠 Quizzy.AI

Quizzy.AI is an AI-powered study companion that transforms your study materials 
into interactive quizzes and flashcards — instantly. Upload your notes, PDFs, 
or documents and let AI do the hard work!

---

## ✨ Features

- 📁 Upload PDFs, documents, or text-based study material
- 🤖 AI automatically generates quizzes & flashcards from your content
- ⚡ Instant results — no manual question creation needed
- 🃏 Flashcard mode for quick revision
- 📝 Quiz mode to test your knowledge
- 🎯 Helps students study smarter and retain information faster

---

## 🚀 How It Works

1. Upload your study material (PDF, DOCX, TXT)
2. AI analyzes and extracts key concepts
3. Get auto-generated quizzes & flashcards in seconds
4. Study, revise, and ace your exams!

---

## 🛠️ Built With

- [Next.js](https://nextjs.org/) — React framework
- [shadcn/ui](https://ui.shadcn.com/) — UI components
- AI Integration — for smart content generation

---

## 💻 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the app.

---

## 🧩 Adding UI Components

To add shadcn components, run:

```bash
npx shadcn@latest add button
```

Components will be placed in the `components/ui` directory.

To use them in your app:

```tsx
import { Button } from "@/components/ui/button";
```
